## 当前系统约束

- `CTA回测` 是单策略单合约单次研究入口
- `Brooks图表` 已支持读取：
  - 单次回测
  - 矩阵回测
  - 优化回测
  - 后续新的 `CTA回测` 自动同步结果
- `Brooks图表` 当前知识点目录按执行流程分类，并开始细化 `背景 -> 突破 / 窄幅通道 / 宽幅通道 / 震荡`
- `Brooks图表` 当前已补：
  - `EMA20` 默认显示，口径按 Al Brooks 默认 `MA = 20 bar EMA`
  - `宽压 / 宽松` 可调图宽
  - `高压 / 高松` 可调图高
  - 右侧价格轴滚轮可直接调图高
  - 下方时间轴滚轮可直接调图宽
  - 下方时间轴改成更短标签，显示更多刻度
- 当前先核对 `背景 -> 突破`
  - 已改成双层模型
  - 底层结构：`震荡 / 窄幅通道 / 宽幅通道`
  - 上层事件：`突破起爆 / 突破跟进 / 突破测试 / 失败突破`
  - 这样可以同时表达结构背景和突破事件，不影响后续策略组合
  - `背景总览` 现改成 Brooks 风格的局部标签/局部框，不再整屏铺色
  - 已去掉无原文支持的短冷却条数规则，改成 breakout mode / 反转次数条件
  - 正常路径优先按 `TR -> BO -> Tight CH -> Broad CH`
- 后续知识点优化顺序：
  1. 背景
  2. 关键位置
  3. setup 前提
  4. signal bar 类型
  5. entry trigger
  6. 触发失效
  7. 初始止损类型
  8. 第一目标
  9. BE 条件
  10. 提前离场
