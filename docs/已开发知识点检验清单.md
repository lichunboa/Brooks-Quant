## 标普数据自查

当前本地“标普 500”相关数据是：

- `US500.OTC`
- 来源：`Dukascopy`
- 类型：`index_cfd`

当前数据质量脚本给出的结果是“通过”，原因是：

- 无重复
- 无异常 OHLC
- 无脚本定义下的可疑缺口

但是从 Brooks 研究用途看，它仍然有一个重要问题：

- 这份 `US500` 是 `CFD` 连续 24 小时数据，最近多日都是 `1440` 根 `1m` bar。
- 这和 Brooks 在课程、百科和 Ali 里最常用的 `Emini S&P 500 / ES`、以及美盘日内交易语境并不完全一致。

结论：

- 这份 `US500 CFD` 可以继续用于程序开发和初步对比。
- 但如果后面要做“直接对照课程截图 / 百科案例 / Ali 日内实战案例”的优化系统，最好补充：
  - `ES` 期货数据，或
  - 至少带 `RTH` 会话模板的指数数据。
- 具体接入建议已整理在：
  [`/Users/mitchellcb/Desktop/量化交易vnpy/docs/ES数据接入选型.md`](/Users/mitchellcb/Desktop/量化交易vnpy/docs/ES数据接入选型.md)
